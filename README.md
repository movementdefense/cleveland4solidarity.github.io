## Set up

### Requirements
- Node    >= 6.9.5
- NPM     >= 3.10.10
- Ruby    >= 2.3.0
- Jekyll  >= 3.3.1
- Bundler >= 1.14.4

### Install

1. Fork this repo to your github account
2. Clone to your computer
3. Run set-up scripts

```bash
bundle install && npm install
```

## Usage

Starting the project
```bash
npm run dev
```
